This repository includes some example files that request data from the [StreamOne API](https://github.com/techdata-cloudautomation/StreamOneIonSDKPython/tree/main).

Feel free to use and modefy it. 

I used it to collect the the following data and saved it as CSV file; Customer	Subscription	Status	Qty	Price	Cost	Margin	Currency	Renewal Date.

### Collect the data and save it as CSV-file.
```
./get_products_simple.sh --csv
```
